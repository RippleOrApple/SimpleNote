# v2.0.0 发布说明

## 版本定位

`v2.0.0` 是简记 SimpleNote 的 V2 Phase 1 版本，重点升级任务工作区、跨平台外观、自适应导航和正文图片能力。

## 主要更新

- 全新的 V2 任务数据模型与 Windows/Android 自适应工作区。
- 支持收集箱、今天、未来七天、全部任务、自定义清单和智能筛选。
- 支持任务搜索、标签、优先级、子任务、排序和完成状态管理。
- Android 使用纯图标底部导航，Windows 使用功能侧栏，并支持导航顺序与默认模块设置。
- 新增预设背景、自定义颜色、字体、密度、模糊、缩放和平台本地外观设置。
- 笔记和任务正文支持本地图片插入、Markdown 预览、全屏查看和事务性删除。
- 支持 Android 图片选择恢复提示，必须显式选择目标笔记或任务，不会自动附加。
- 旧版同步协议无法识别 V2 数据，生产环境已停用 V1 同步入口；V2 同步将在 Phase 4 提供。

## 验证结果

- `flutter analyze` 通过，无静态分析问题。
- `flutter test` 通过，共 186 条测试。
- Windows Release 构建通过。
- Android Release APK 构建通过。
- Android 模拟器启动、Today 首屏、纯图标导航及同步升级提示检查通过。

完整记录见 [V2 Phase 1 验收文档](V2_PHASE_1_ACCEPTANCE.md)。

## 安装说明

- Windows：下载 ZIP，完整解压后运行 `simple_note.exe`。
- Android：下载 APK 后安装。当前 APK 使用项目的调试签名，适合测试和演示；应用商店发布前需要配置正式签名。

## 已知限制

- V2 局域网同步将在 Phase 4 实现，当前版本保持本地离线可用。
- Android 相机、图库、真实触觉和选择器进程恢复仍建议在物理设备上完成发布前检查。
- Windows 干净构建环境需要启用 Developer Mode 或提供符号链接权限。
